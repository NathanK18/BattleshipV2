Nathan Kitchens Battleship Exam 1 

This is a browser based Battleship game with JavaScript as the front-end and PHP backend.

To run this game you need to place the Battleship folder in your XAMPP using the path /Applications/XAMPP/xamppfiles/htdocs/Battleship/

Then run XAMPP and access the game using the URL http://localhost/Battleship/
